Étape 1 : Choix des composants

Je suis parti d'une base pour du gaming 1080p léger.

  • CPU : I5 12400
  • GPU : GTX 1650LP
  • RAM : 16GB DDR4 3200MHz
  • Stockage : SSD NVMe 250GB + HDD 1TB
  • Alimentation : 550W 80+ Gold
  • Boîtier : Zalman T6 ATX
    •

    Étape 2 : Montage du boîtier

    Suite à des pannes de composants, j'ai dû changer de RAM, de carte graphique et de CPU. Je me suis aussi procuré un nouveau boîtier ainsi qu'un watercooling 240 mm.

  • CPU : I5 12600KF
  • GPU : GTX 1650
  • RAM : 16GB DDR4 3600MHz
  • Stockage : SSD NVMe 750GB + HDD 1TB
  • Alimentation : 550W 80+ Gold
  • Boîtier : AeroCool Shard-G-BK
    •

    Étape 3 : Installation CPU/RAM

    Pour être plus à l'aise avec les performances, j'ai opté pour plus de RAM et une nouvelle carte graphique. Pour l'esthétique, j'ai choisi un boîtier blanc avec une bonne ventilation.

  • CPU : I5 12600KF
  • GPU : RTX 3050 8GB
  • RAM : 32GB DDR4 3600MHz (2x8 Corsair Vengeance, 2x8 Kingston Fury Beast)
  • Stockage : SSD NVMe 750GB + HDD 1TB
  • Alimentation : 850W 80+ Gold
  • Boîtier : M.RED Cobalt Blanc
    •

    Étape 4 : Câblage

    Suite a la pane de deux de mes barres de RAM, je les remplacer par 2 Nouvelles barres de 16GB puis j'ai ajouté du stockage.

  • CPU : I5 12600KF
  • GPU : RTX 3050 8GB
  • RAM : 32GB DDR4 3600MHz (2x16 Corsair Vengeance)
  • Stockage : SSD NVMe 1To + HDD 1TB + SSD SATA 250GB
  • Alimentation : 850W 80+ Gold
  • Boîtier : M.RED Cobalt Blanc
    •

    Étape 5 : Optimisation BIOS

    La dernière modification a été de changer la 3050 qui devenait limite en gaming pour une RTX 5060.
