Hasher - Générateur de mot de passe sécurisé

App Web locale (hors ligne) pour créer un mot de passe unique par service à partir d'un mot de passe maître.

JavaScript Crypto API Frontend UX

Utilisez le projecteur ci-dessous pour tester l’application sans quitter la page, puis consultez les parties clés du code et l'explication.

1. Fonctionnalité principale

  • Entrées: mot de passe maître + service.
  • Hash sécurisé via crypto.subtle.digest('SHA-256', ...).
  • Transformation du hash en mot de passe de 16 caractères.
  • Pas de stockage spécifique côté serveur; tout reste local.

2. Extrait de code

async function generate() {
    const master = document.getElementById("master").value;
    const service = document.getElementById("service").value;

    if (!master || !service) {
        alert("Remplis les champs");
        return;
    }

    const text = master + "::" + service;
    const encoder = new TextEncoder();
    const data = encoder.encode(text);

    const hashBuffer = await crypto.subtle.digest("SHA-256", data);
    const hashArray = Array.from(new Uint8Array(hashBuffer));

    const chars = "abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789!@#$%^&*()_+";
    let password = "";

    hashArray.forEach(byte => {
        password += chars[byte % chars.length];
    });

    document.getElementById("result").value = password.slice(0, 16);
}

3. Explication

Le code génère une chaîne unique par service via un hachage SHA-256 du couple master + service. Ce hachage est mappé sur un ensemble de caractères sécurisé avant d’être tronqué à 16 caractères pour former le mot de passe.

Ce pattern donne un mot de passe reproductible (pour le même master+service) tout en évitant de les stocker (via la crypto native du navigateur).
