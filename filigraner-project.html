Filigraner - Générateur de filigranes pour images

App Web locale pour ajouter des filigranes personnalisés sur vos images et PDFs, entièrement traitée côté client.

JavaScript Canvas API PDF.js Frontend

Utilisez l'application ci-dessous pour tester la génération de filigranes en temps réel, puis consultez les parties clés du code.

1. Fonctionnalités principales

  • Support des images (JPG, PNG, etc.) et des PDFs.
  • Personnalisation complète du filigrane: texte, taille, angle, opacité.
  • Modes: filigrane centré ou répété sur toute l'image.
  • Styles variés: normal, gras, italique, avec ombre.
  • Téléchargement direct de l'image avec filigrane (PNG).
  • Traitement 100% local, aucune donnée n'est envoyée.

2. Extrait de code - Fonction de génération

function generateWatermark() {
    if(!uploadedImage) {
        alert('Veuillez d\'abord sélectionner une image');
        return;
    }
    
    const canvas = document.getElementById('canvas');
    const ctx = canvas.getContext('2d');
    
    canvas.width = uploadedImage.width;
    canvas.height = uploadedImage.height;
    
    ctx.drawImage(uploadedImage, 0, 0);
    
    const text = document.getElementById('watermarkText').value;
    const angle = document.getElementById('angle').value;
    const fontSize = document.getElementById('fontSize').value;
    const opacity = document.getElementById('opacity').value / 100;
    const style = document.getElementById('style').value;
    
    ctx.save();
    ctx.globalAlpha = opacity;
    ctx.font = `${style === 'bold' ? 'bold ' : ''}${fontSize}px Arial`;
    ctx.fillStyle = 'rgba(255, 255, 255, 0.5)';
    
    // Mode répété ou centré
    const mode = document.getElementById('mode').value;
    if(mode === 'repeat') {
        // Boucle sur toute l'image
        const spacing = 100;
        for(let x = 0; x < canvas.width; x += spacing) {
            for(let y = 0; y < canvas.height; y += spacing) {
                ctx.save();
                ctx.translate(x, y);
                ctx.rotate((angle * Math.PI) / 180);
                ctx.fillText(text, 0, 0);
                ctx.restore();
            }
        }
    } else {
        // Centré
        ctx.translate(canvas.width / 2, canvas.height / 2);
        ctx.rotate((angle * Math.PI) / 180);
        ctx.textAlign = 'center';
        ctx.fillText(text, 0, 0);
    }
    
    ctx.restore();
}

3. Explication technique

L'application utilise Canvas API pour dessiner le filigrane dynamiquement sur l'image. Les étapes principales sont:

  • Chargement: Les images sont lues via FileReader et les PDFs via PDF.js
  • Rendu: L'image est dessinée sur le canvas, puis le filigrane est superposé
  • Transformation: Rotation, opacité, styles appliqués via le contexte 2D du canvas
  • Mode répété: Boucles imbriquées pour couvrir toute la surface
  • Export: Le canvas est converti en PNG via toDataURL()

Le traitement se fait entièrement dans le navigateur, ce qui garantit la confidentialité totale des documents.
